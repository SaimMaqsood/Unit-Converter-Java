# Unit-Converter-Java
A simple console-based program that converts between different units of measurement.

### What it does
- Converts length units (cm, m, km, inch)
- Converts temperature units (Celsius, Fahrenheit, Kelvin)
- Takes user input for category, value, from unit, and to unit
- Shows the converted result with 2 decimal places
- Includes basic error messages for unknown units

### How to run
1. Make sure you have Java installed
2. In the project folder open terminal or command prompt
3. Compile all files:
4. Run the program:
Java UnitConverterTester
### Example usage
Choose category: 1  
Enter value: 100  
From unit: cm  
To unit: inch  
→ 100.00 cm = 39.37 inch

### Files
- **UnitConverter.java**  
Contains the conversion logic for length and temperature (can be extended for more types).

- **UnitConverterTester.java**  
Interactive console program: asks for category, value, from/to units, and displays result.
